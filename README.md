# 📌 Problemas e Soluções Digitais

Projeto simples desenvolvido por mim como proposta de desafio acadêmico/prático utilizando HTML, CSS e JavaScript.

Este foi um dos meus projetos como iniciante na área de desenvolvimento web, com foco em praticar lógica de programação, interatividade e manipulação de elementos na página.

O objetivo do site é reunir pequenas ferramentas e exercícios matemáticos em uma única aplicação simples e funcional.

---

## 🚀 Funcionalidades

O sistema possui:

- Botão interativo
- Cálculo da área do triângulo
- Verificação lógico-matemática
- Descoberta do maior número
- Conversor de temperatura
- Calculadora de investimento
- Verificação de divisibilidade
- Descoberta de meses
- Reajuste salarial
- Cálculo de combustível
- Conversão de minutos
- Calculadora de gorjeta
- Simulação de empréstimo
- Desconto previdenciário
- Simulação de rendimento financeiro

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
